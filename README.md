# 뭐마실뀨
## 소개
> 시각장애인 전용 음료수 분석 iOS Application
1. 스마트폰으로 음료 사진을 찍으면 AI가 음료를 분석하여 음료 이름과 정보를 제공합니다.
2. 고채도 메인 컬러, 큰 아이콘 크기, 최소한의 중요 정보 표시 등 시각장애인인 사용자의 특성을 고려하여 UI를 구성하였습니다.

## 주요 기능
- 사용자 맞춤 사진 촬영 기능
- AI 음료수 분석 기능
- 음료 이름 및 세부 정보 제공 기능

## 개발 환경
- Xcode @12.5
- Swift 5

## Application Version
- minSdkVersion: 14.5
